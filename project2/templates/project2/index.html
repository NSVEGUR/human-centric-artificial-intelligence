What are Counterfactual Explanations?

Counterfactuals answer the question: "What would need to change for the model to predict a different class?" They help you understand what minimal changes to a penguin's measurements would result in it being classified as a different species.

Method: We sample many points near the original, find those predicted as the target class, and rank them by MAD-weighted L1 distance (smaller = more similar to original).

Method reference How PDP & ALE work ▼

PDP (Partial Dependence Plot)

Shows the average effect of a feature on the prediction. For each feature value, we average the predictions over all samples.

ALE (Accumulated Local Effects)

More robust when features are correlated. Computes local effects by looking at how predictions change within small intervals.

Method: finite difference (discretization)
