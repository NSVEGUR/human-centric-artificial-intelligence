Project 3 — Active Learning for Learning-to-Defer

Human-Centric Artificial Intelligence · AG News Dataset

This project trains a classifier on the AG News dataset and simulates human experts with domain-specific strengths. The goal is to learn when to defer decisions to a human expert using active learning.

Task 1

Baseline Classifier — TF-IDF + Logistic Regression

Trained on the full AG News training set (120,000 articles, 4 categories). Features: TF-IDF with bigrams (50k features). Model: Logistic Regression (C=5).
